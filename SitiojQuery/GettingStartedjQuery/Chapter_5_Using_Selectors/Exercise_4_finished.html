Selecting Elements by Position

jQuery uses more CSS selectors that allow you to select an element by its position. For example, you can use the > selector to select elements that are direct children of other elements.

See the chart below for jQuery Supported CSS selectors.

CSS calls selectors such as button:down pseudo-selectors. jQuery calls these selector types filters.
