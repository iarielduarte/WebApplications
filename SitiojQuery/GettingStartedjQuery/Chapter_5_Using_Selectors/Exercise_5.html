Using jQuery Selectors

In addition to CSS Selectors, jQuery supports many robust custom filter selectors.

Below is an incomplete list of jQuery selectors (some may be included in CSS as well)

Selector Description Included in CSS
:animated Selects elements that are controlled with animation
:button Selects only button elements
:checked Selects checkboxes or radio buttons that are in a checked state
:not(selector Selects any elements that are not of the selector type.
:password Selects only password elements
:disabled Selects only elements in a disabled state
:enabled Selects only elements in an enabled state
:contains(T) Selects only elements containing the text "T"
:has(selector) Selects only elements that contain at least one element that matches the selector
