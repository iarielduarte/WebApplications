Side Note

hover(): Bind two handlers to the matched elements, to be executed when the mouse pointer enters and leaves the elements.

The .hover() method binds handlers for both mouseenter and mouseleave events. You can use it to simply apply behavior to an element during the time the mouse is within the element, because it only fires the event when the area covered by the elements is hovered and ignores transitions to child elements.

mouseover(): Bind an event handler to the "mouseover" JavaScript event, or trigger that event on an element.

The mouseover event is sent to an element when the mouse pointer enters the element. Any HTML element can receive this event.
