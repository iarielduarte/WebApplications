Understanding jQuery Fundamentals

The jQuery syntax is designed for selecting HTML elements and then performing some action on the selected element or elements.

Basic syntax is: $(selector).action()

  • A dollar sign is used to define the jQuery
  • A (selector) to "query (or find)" HTML elements
  • A jQuery action() to be performed on the discovered (found) element or elements
  1. Create an HTML page or use this one.
  2. Create the link to the jquery library:
    <script src="../scripts/jquery_1_6_1.js"></script>

  3. Create a <script> block and call the jQuery function using the syntax:
    $(selector)

    The selector is similar to CSS selectors and XPath selectors in that it selects one or more elements in the HTML document. The selector will be added in step #7.
    This is an alias to the jQuery function: jQuery() which may also be used here.
    This function returns a JavaScript object that contains an array of the HTML files DOM elements in the order in which they are defined in the HTML

  4. Create a <div> tag with content or use the #jQuerySummary div below.
  5. Create two buttons with the labels "Show Summary" and "Hide Summary".



  6. Locate the <div> with jQuery:

    $('jQuerySummary')

  7. Invoke the show() method of the jQuery Wrapper object:

    $('jQuerySummary').show();

  8. Place the code in the document ready block:

    <script>
    $(document).ready(function(){
    $("showSummary").click(function(){
    $("#jQuerySummary").show();
    });
    });
    </script>

  9. Repeat the code and modify it to invoke the hide() method.

    <script>
    $(document).ready(function(){
    $("hideSummary").click(function(){
    $("#jQuerySummary").hide();
    });
    });
    </script>

jQuery() Summary

jQuery() Wrapper

  • Uses Selectors like CSS and XPath
  • jQuery() returns an object (the jQuery wrapper) containing an array of DOM elements
  • The wrapper object contains a large number of useful methods that act on the collected group of elements returned by the selector
