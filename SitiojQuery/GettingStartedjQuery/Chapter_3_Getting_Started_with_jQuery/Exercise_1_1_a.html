Alternative methods for using the jQuery Library?

Instructions:

  1. Create your web site or web application directories.
  2. Use the jQuery library that is hosted on Google Code
  3. Create a sample web page or use this one.
  4. Write a script tag whose src attribute references the jQuery library on Google code

    <script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.4.2/jquery.min.js"></script>

  5. Write a script tag whose src attribute references the jQuery library on Microsoft code:

    <script type="text/javascript" src="http://ajax.microsoft.com/ajax/jquery/jquery-1.4.2.min.js"></script>

Congratulations!

You are now ready to being using code in the jQuery Library!
