How do I use the jQuery Library?

Instructions:

  1. Create your web site or web application directories.
  2. Create a folder called "scripts" in your website/application.
  3. Download the jQuery library from: Click Here (for http://jquery.com/) and place the code in the folder you created in step two.
  4. Create a sample web page or use this one.
  5. Write a script tag whose src attribute references the jQuery library

    <script src="scripts/jquery.js"></script>

Congratulations!

You are now ready to being using code in the jQuery Library!
