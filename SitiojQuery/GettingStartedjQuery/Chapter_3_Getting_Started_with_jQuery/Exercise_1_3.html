How do I begin to execute jQuery?

Instructions:

With JavaScript we wait for a page to load before we execute our JavaScript statements (as shown below).

window.onload = function(){ alert("welcome"); }

With jQuery we wait for the jQuery ready event.

.ready()

Specify a function to execute when the DOM is fully loaded.

  1. Create a new html file or use this one.
  2. Be sure the file you create contains a hypertext link (as shown above in the heading 2 link .ready() to http://api.jquery.com/ready/)
  3. Create a script block.
  4. Create a function to be executed after the document ready event
     <script>
       $(document).ready(function()
			{         
				$("a").click(function(event)
					{ 
			          alert("You just prevented the default action with 'event.preventDefault();'");
					  event.preventDefault();
				    }
		    }

      </script>

The jQuery syntax is designed for selecting HTML elements and then performing some action on the selected element or elements.

Basic syntax is: $(selector).action()

  • A dollar sign is used to define the jQuery
  • A (selector) to "query (or find)" HTML elements
  • A jQuery action() to be performed on the discovered (found) element or elements
