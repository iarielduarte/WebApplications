Looping through a wrapped set

jQuery provides methods for iterating through a wrapped set or a portion of a wrapped set.

The each(iterator) method accepts a function argument that contains two parameters. The first is a zero-based index of the element within the set and the second is the element itself.

Example:

$('img').each(function(n, element) {

   this.title = 'This is picture # [' + n + ']. The id is ' + this.id;

});
