Adding Elements to sets and working with jQuery Chains

jQuery provides many methods for selecting portions of a wrapped set. These methods include the following:*

* not a complete list.

Method Description
get(index) Returns the element at the index in the wrapped set.
eq(index) Returns a single element at the index in the wrapped set.
first() Returns the first element in the wrapped set.
last() Returns the last element in the wrapped set.

After creating a wrapped set of elements, you can add to the set with the add() method.

Between the creation of the two sets you can perform actions.
